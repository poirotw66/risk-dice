# 🎨 前端頁面優化完成

已成功優化風險骰子前端頁面，大幅提升沈浸感和遊戲氛圍。

## ✨ 新增功能

### 1. 📖 可展開的骰子說明區塊

在頁面頂部標題下方新增「查看說明 / 隱藏說明」按鈕，點擊後展開完整的風險骰子說明：

**包含內容：**
- ⚠️ 風險骰子背景介紹（《獵人》設定）
- 🎲 外型與結構說明（D20，19面大吉 / 1面大凶）
- 🎯 擲骰規則與本質
- ⚠️ 真正的風險警告
- 🧠 道具哲學
- 🩸 使用警告

### 2. 🎭 視覺設計增強

#### 說明面板設計
- **配色方案**: 深灰到暗紅漸層背景
- **邊框效果**: 紅色發光邊框 (`border-red-900/50` + `shadow-[0_0_30px_rgba(139,0,0,0.5)]`)
- **分區設計**:
  - 黃色左側邊框區塊（基本介紹）
  - 綠色邊框（大吉說明）
  - 紅色邊框（大凶警告）
  - 深紅背景警告框（使用警告）

#### 文字排版
- 使用 `VT323` 等寬字體強化復古遊戲風格
- 行高 `1.8` 提升可讀性
- 多層級標題結構
- 強調色彩搭配：
  - 🟦 青色：命運、機制說明
  - 🟪 粉色：運氣、玩家行為
  - 🟨 黃色：關鍵數據
  - 🟥 紅色：危險、警告

### 3. 🎬 動畫效果

#### 新增 slideDown 動畫
```css
@keyframes slideDown {
  from {
    opacity: 0;
    transform: translateY(-20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```

**效果**: 說明面板展開時從上方滑入，持續 0.3 秒

### 4. 💡 互動設計

#### 展開/收合按鈕
- 位置：標題卡片底部
- 樣式：半透明青色，懸停變亮
- 圖示：▼ / ▲ 箭頭指示狀態
- 文字：「查看說明」 / 「隱藏說明」
